# Codex 작업 규칙

## 논문 리뷰 / 기사 요약 → Notion 저장

- 논문 정리, 리뷰, 요약 요청이 오면 **텍스트로 답변하지 말고** 반드시 `mcp__notion_summary_server__save_summary_to_notion` 도구로 Notion에 저장한다.
- 명시적으로 "텍스트로 알려줘"라고 한 경우만 예외.

## Notion 콘텐츠 포맷

- `##`, `###` 같은 마크다운 헤더 사용 금지.
- 섹션 구분은 `━━━ 섹션명 ━━━` 형식 사용.
- 짧은 bullet point 나열이 아닌 문단 형식으로 충분히 서술 (배경, 가설, 실험, 결과, 의의, 한계, 질문 등 포함).
- 이모지 적극 사용 — 섹션 구분선, 핵심 키워드, 수치 강조 등에 자연스럽게 포함한다.

## 다이어그램

- 논문 구조나 개념을 시각적으로 표현하면 이해에 도움이 될 때, ASCII/유니코드 텍스트 다이어그램을 리뷰 본문에 포함한다.
- 단순한 내용이거나 다이어그램이 필요 없는 논문이면 생략해도 된다.
- 다이어그램은 별도 섹션으로 만들지 않고 관련 설명 섹션 안에 자연스럽게 포함한다.

## 개념/용어 설명

- 논문 이해에 필요한 배경 개념이나 용어 설명은 별도 페이지로 만들지 않고, 리뷰 본문 페이지에 섹션으로 통합한다.
- 추후 추가 요청이 오면 기존 페이지를 업데이트한다.

## Notion DB 라우팅

- `category="paper"` → AI 논문 DB (`34617b872be68060a474e18a73510f38`)
- `category="stock_research"` → 주식 리서치 DB (`39bf50a1aca04ad2a10079e958cf96d4`) — 뉴스 / 산업분석 / 주간브리핑
- `category="stock_study"` → 주식 공부노트 DB (`d1afcb876857487eb978c1a8e0952d05`) — 용어 / 이론 / 지표 / 전략 / 산업지식

모두 "Finance & 투자" 페이지(`33117b872be68187a1b4ddc51261856e`) 인라인 DB.

## 주식 리서치 저장 규칙

- `sub_category`: 뉴스 | 산업분석 | 주간브리핑
- `source`: 출처 (예: "한국경제", "연합뉴스")
- `tags`: 관련 섹터/종목 (예: ["반도체", "AI", "국내"])
- `icon`: 페이지 아이콘 이모지 (예: "📰", "📊", "🔍")
- `difficulty`: stock_study 전용 — 기초 | 중급 | 고급

## 주식 콘텐츠 포맷

- 논문과 동일하게 `##`, `###` 헤더 금지. `━━━ 섹션명 ━━━` 형식 사용.
- 뉴스/리포트: 핵심 요약 → 시장 영향 → 투자 시사점 순으로 서술.
- 공부노트: 개념 정의 → 실제 사례 → 투자 적용법 순으로 서술.
- 단순 bullet 나열 금지, 문단 형식으로 충분히 서술.
- 이모지 적극 사용 — 섹션 구분선, 수치, 추천/주의 표시 등에 자연스럽게 포함한다.

## 일일 자동화 (daily.py)

매일 아래 명령어로 실행. 오늘 날짜 페이지가 이미 있으면 자동 스킵.

```powershell
& "D:\Anaconda3\envs\project\python.exe" "D:\projects\python\notion_summary_server\daily.py"
```

### 자동 저장 항목 3개

| 항목 | 제목 | 아이콘 |
|------|------|--------|
| ① 뉴스 종합 | `[날짜] 경제 뉴스 종합` | 📰 |
| ② 주간브리핑 | `[날짜] 주간 투자 브리핑` | 📊 |
| ③ Claude 인사이트 | `[날짜] Claude 인사이트` | 🔍 |

### ③ Claude 인사이트 데이터 소스

- 연합뉴스 RSS (국내 경제)
- 한국경제·매일경제 RSS (국내 경제지)
- Reuters·CNBC·MarketWatch·연합뉴스 국제 RSS (해외)
- 네이버 금융 리서치 (증권사 리포트 제목·날짜)
- yfinance 실시간 시장 데이터 (KOSPI·KOSDAQ·S&P500·NASDAQ·달러인덱스·WTI·금·달러/원)

## 주간 브리핑 프로토콜

"주간 브리핑", "이번 주 투자 추천", "시장 브리핑" 등의 요청이 오면 아래 순서로 수집·분석 후 Notion에 저장한다.

수집 항목:
1. 글로벌 거시경제 — 미국 금리(연준 동향), 달러인덱스, 유가(WTI), 금 가격
2. 미국 시장 — S&P500 / 나스닥 주간 등락, 주요 ETF(QQQ·SPY·SOXX 등), 핵심 뉴스
3. 국내 시장 — KOSPI / KOSDAQ 주간 동향, 외국인·기관 수급, 주요 이슈
4. 유망 섹터 분석 — 현재 모멘텀 상위 섹터 2~3개, 이유 설명
5. ETF/종목 추천 — 섹터별 대표 ETF 또는 종목, 투자 근거 및 리스크

저장 방법:
- `category="stock_research"`, `sub_category="주간브리핑"`
- `title`: "YYYY-MM-DD 주간 투자 브리핑"
- `tags`: 해당 주 핵심 섹터 태그 포함
- `source`: "Codex 자동 수집"
